Detailed Description

An iterator over the directory entry structures in a FAT filesystem.

A directory entry iterator iterates over the 32 byte DirectoryEntry structures in a FAT directory. A base class is defined here so that the special case of iterating over the FAT16 root directory is abstracted.

enum stm32plus::fat::DirectoryEntryIterator::Options

Options bitmask for the iteration

Enumerator:
OPT_NONE 

No options.

OPT_SKIP_DELETED 

Entries that are marked as deleted will be skipped over.

OPT_PARSE_LONG_NAMES 

Long names are spread over multiple directory entries. Set this option to parse them as one entry.

OPT_DEFAULT_REAL_ENTRIES 

Default options are to skip deleted and parse long names.

Move to the next item in the sequence, or on to the first item in the sequence if this is the first call to next().

Write some new dirents to this directory. If the directory contains a contiguous sequence of deleted large enough to hold the new sequence then these entries are used. If not then new entries are allocated on the end of the directory. Whichever happens, the new dirents are guaranteed not to be broken up.
