Detailed Description

template<uint8_t TDmaNumber, uint8_t TChannelNumber>
class stm32plus::DmaInterruptFeature< TDmaNumber, TChannelNumber >

Base class for the DMA feature to handle interrupts. This class is the only Observable in the DMA class suite.

Template Parameters
TDmaNumberThe number of the DMA peripheral (1..2)
TChannelNumberThe number of the DMA channel (1..7) or (1..5)
