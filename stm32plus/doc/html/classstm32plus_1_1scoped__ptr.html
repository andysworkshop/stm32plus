Detailed Description

template<class C>
class stm32plus::scoped_ptr< C >

A scoped_ptr<T> is like a T*, except that the destructor of scoped_ptr<T> automatically deletes the pointer it holds (if any). That is, scoped_ptr<T> owns the T object that it points to. Like a T*, a scoped_ptr<T> may hold either nullptr or a pointer to a T object. Also like T*, scoped_ptr<T> is thread-compatible, and once you dereference it, you get the threadsafety guarantees of T.

The size of a scoped_ptr is small: sizeof(scoped_ptr<C>) == sizeof(C*)
