Detailed Description

Pass-through touch-screen calibration.

This class gives access to the raw co-ordinates from the touch screen by passing them through unmodified. This is typically used by a calibration routine to avoid the chicken-and-egg situation of the TouchScreen class requiring a calibration class even before it's calibrated.
