Detailed Description

Base feature class to enable polling communication in master mode. This class manages the actual I2C protocol interaction. It makes the assumption that the exchange with a peripheral consists of sending N bytes as an "address" and then either reading or writing some data. The address size (N) can be 1..255. That size can be found in the I2C Parameters class.

Prepare for a read from the slave. This will do the I2C protocol up to the point of sending the address bytes to read from. After the successful completion of this function you are expected to complete the read either by polling or by DMA using the I2CDmaReaderFeature class.
