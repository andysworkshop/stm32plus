Detailed Description

template<typename T>
class stm32plus::Iterator< T >

Iterator base class.

Template Parameters
TThe type that we are iterating over.

Iterators are created pointing into nothing before the first item. A call to next() is required to move to the first item and on through the items until next() returns false. An error may also cause next() to return false. The error provider should be checked to differentiate between end-of-sequence and fail.
