Detailed Description

template<uint16_t TInputTrigger, uint16_t TSlaveMode>
class stm32plus::TimerSlaveFeature< TInputTrigger, TSlaveMode >

Feature class to configure this timer as a slave for another timer. The slave mode and input trigger (0..3) are configurable.
