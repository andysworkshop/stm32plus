Detailed Description

template<class TPeripheralInfo, uint16_t TTimerEvent, uint32_t TPriority = DMA_Priority_High, uint32_t TDmaMode = DMA_Mode_Normal>
class stm32plus::TimerDmaFeature< TPeripheralInfo, TTimerEvent, TPriority, TDmaMode >

Template definition of a DMA feature used for transferring data to a peripheral where that transfer is regulated by events from a timer.

Template Parameters
TPeripheralInfoa helper class used to describe key data about the target peripheral
TTimerEventThe timer event that triggers a transfer, e.g. TIM_DMA_Update
TPriorityThe DMA priority, default is high
TDmaModecircular or normal mode (DMA_Mode_Normal is the default)
