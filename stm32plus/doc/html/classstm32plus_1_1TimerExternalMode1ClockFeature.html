Detailed Description

template<uint16_t TTriggerSource>
class stm32plus::TimerExternalMode1ClockFeature< TTriggerSource >

feature class to configure the timer as using an external clock as a trigger. The counter can count at each rising or falling edge on a selected input.

The user must call setClock() on the the timer object.
