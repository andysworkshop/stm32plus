Detailed Description

Buffered IO stream operating as a circular buffer.

The buffer is non-blocking and it will not overwrite data that has not been read. The read and write functions will return errors if the buffer is full. Read functions will set E_END_OF_STREAM when there is no data available. Write functions will return E_BUFFER_FULL if there is not enough space for the write.

This is designed to be safe for using with interrupts. e.g. an ISR writes to to the class and the main code reads from it, or vice-versa. The write() and read() functions do not share any internal state members and all state members are <= the native word size of the MCU.
