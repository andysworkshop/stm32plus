Detailed Description

This is the primary include file for stm32plus. You should always include it before you include any other config files.

Configuration for the library is here. Make sure you have defined one of the MCU selection defines:

STM32PLUS_F1_HD STM32PLUS_F1_CL (not yet fully tested) STM32PLUS_F4

Also ensure that HSE_VALUE has been set to the speed of your external oscillator in hertz. The scons build procedure takes care of all these definitions.
