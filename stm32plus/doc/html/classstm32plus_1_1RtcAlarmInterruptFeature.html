Detailed Description

template<uint32_t TAlarmIndex>
class stm32plus::RtcAlarmInterruptFeature< TAlarmIndex >

RTC feature to enable access to the alarm functionality

RTC feature to enable access to the alarm functionality. The F4 has two independent alarms that can be configured to trigger on any or all of the time components and optionally the date/weekday.
