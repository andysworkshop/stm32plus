Detailed Description

template<class TFsmc>
class stm32plus::FsmcSram< TFsmc >

FSMC implementation for SRAM memory.

The memory is assumed to be connected in the manner documented by ST in AN2784. The STM32 package is assumed to be 144 pin and the memory transfers are 16-bit.
