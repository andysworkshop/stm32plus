Detailed Description

File information base class.

Defines the valid file attributes and the methods that a derived class must supply.

Member Enumeration Documentation

enum stm32plus::FileInformation::FileAttributes

Abstract file attributes. Full set may not be supported by all file systems. These attributes are intended to match FAT attributes 1:1 and will always do so.

Enumerator:
ATTR_READ_ONLY 

File is read only.

ATTR_HIDDEN 

File is hidden.

ATTR_SYSTEM 

File is a system file.

ATTR_VOLUME_ID 

File is actually the volume label.

ATTR_DIRECTORY 

File is a directory.

ATTR_ARCHIVE 

File has been archived.
