Detailed Description

template<class TImpl>
class stm32plus::SerialEeprom< TImpl >

Template implementation of a serial EEPROM. Inherits from InputStream and OutputStream. Provides the functionality for maintaining the stream pointer. Delegates the actual read and write operations to the TImpl class parameter

Template Parameters
TImplThe device implementation (this is the CRTP template pattern)
