Detailed Description

base class for touch screen calibration algorithms.

This base class enforces an API on derived classes but does not hide the type of calibration algorithm since the user must still choose this at runtime.

virtual bool stm32plus::display::TouchScreenCalibrator::calibrate ( TouchScreenCalibration *&  calibrationResults)
pure virtual

Interactively calibrate the touch screen with the user. This routine will present a UI to the user for calibration and block until it's done. The resulting calibration class will be returned in the calibrationResults parameter and can be serialised to an output device for restoration in future sessions.
