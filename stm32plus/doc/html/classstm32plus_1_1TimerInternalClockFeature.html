template<uint8_t TAPBBusNumber>
class stm32plus::TimerInternalClockFeature< TAPBBusNumber >

Feature class to configure the timer as using the internal clock as its source.

Constructor & Destructor Documentation

Specialisations for APB1 and APB2. The bus prescaler and the not very well documented x2 multiplier that applies when the prescaler is not 1 are taken into account when setting the timer clock.

TIM1,8,9,10,11 are on APB2. The remainder are on APB1
