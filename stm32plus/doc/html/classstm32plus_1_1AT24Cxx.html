Detailed Description

template<class TI2C, int TSizeInBytes>
class stm32plus::AT24Cxx< TI2C, TSizeInBytes >

Template class that defines the Atmel AT24C32/64 interface.

The Atmel AT24C32/64 is a 32768/65536 kbit serial EEPROM IC with an I2C interface. Data can be read or written using single-byte random access or it can be accessed serially in multiples of up to 32 bytes (a page). That is, you may access from where you start up to the end of a page in one burst and then you must update your position to the next page and may then read/write up to another 32 bytes in a burst. This class takes care of all that internal stuff for you.
