Detailed Description

template<uint8_t TTimerNumber>
class stm32plus::TimerInterruptFeatureEnabler< TTimerNumber >

Helper class to enable only the desired interrupts in the NVIC. This will be fully specialised for each timer

Template Parameters
TTimerNumberThe number of the timer peripheral
