Detailed Description

template<typename T>
class stm32plus::Memblock< T >

Template class to manage a block of data on the heap.

Template Parameters
TThe type being managed here.

Assignment makes a copy and does not transfer ownership of the original. Therefore a=b will make a new copy of b. Memory is copied from b to a using memcpy so obviously this only works with simple types and structures.
